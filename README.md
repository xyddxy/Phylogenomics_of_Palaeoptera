# Phylogenomics confirms Palaeoptera monophyly and illuminates the origin of winged insects

Shiyu Du, Yehao Wang, Feng Zhang, Michael S. Engel, Xuhongyi Zheng, Dewen Gong, Changfa Zhou, Chenyang Cai

## Here is a breakdown of the main files uploaded to this repository:

Appendix A.pdf: All 18 phylogenetic trees.

Appendix B.zip: All treefiles, and log files and runtrees files of phylogenetic trees under '--bnni' and '--run10' parameters with ML analyese using IQ-TREE.

Appendix C.xlsx: The alpha value of the Gamma shape distribution, amino acid substitution frequencies, sequence lengths, and the proportion of saturated sites for each gene extracted from empirical datasets.

Appendix D.zip: Detailed θ values of Gene tree simulation.

Appendix E.docx: Commands and parameters used for gene tree simulation.

Appendix F.txt: Detailed simulation commands for 10,000 gap-free AA alignments.

Appendix G.pdf: All three independent runs of divergence-time trees.

Appendix H.nexus: Nexus file used for FcLM analysis.

treefile.zip: All 21 treefiles of phylogenetic trees and MCMCTrees.
